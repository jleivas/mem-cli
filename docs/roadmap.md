# Roadmap

## MVP 1

- CLI installable on Linux, macOS, and Windows
- `start`, `stop`, `status`, `dashboard`, and `version`
- live token dashboard using simulated data

## MVP 2

- adapters for real token events from agent sources
- generic event ingestion interface
- better runtime diagnostics

## MVP 3

- persistent memory per project
- local read/write memory operations
- project-scoped state and indexing

## MVP 4

- local MCP server
- memory and observability exposed through MCP tools
- safe local service lifecycle controls

## MVP 5

- multiplatform distribution artifacts
- installers and packaging improvements
- plugin and adapter ecosystem
